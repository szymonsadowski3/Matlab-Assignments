ad1) Analiza liniowego równania różnicowego przy n = 1

x(k+1) = lambda*x(k)

Równanie charakterystyczne:

z - lambda = 0

Ma jedną wartość własną: lambda

a) lambda < -1

Wówczas system będzie niestabilny, bowiem wartość własna leży poza kołem jednostkowym. Również oscylujący, gdyż wartość własna jest ujemna

b) lambda = -1

Wówczas system będzie jedynie stabilny, bowiem wartość własna leży na brzegu koła jednostkowego. Również oscylujący, gdyż wartość własna jest ujemna. Zauważmy, że x(k) = x(0) dla k nieparzystego; x(k) = - x(0) dla k parzystego.

c) lambda w (-1;0)

Wówczas system będzie asymptotycznie stabilny, bowiem wartość własna leży wewnątrz koła jednostkowego. Również oscylujący, gdyż wartość własna jest ujemna.

d) lambda = 0

Wówczas x(k) jest tożsamościowo równe 0. Układ zawsze jest w stanie równowagi

e) lambda w (0;1)

Wówczas system będzie asymptotycznie stabilny, bowiem wartość własna leży wewnątrz koła jednostkowego. Brak oscylacji.

f) lambda = 1

Wówczas system będzie jedynie stabilny, bowiem wartość własna leży na brzegu koła jednostkowego. Brak oscylacji. Zauważmy, że x(k) jest tożsamościowo równe x(0).

g) lambda > 1

Wówczas system będzie niestabilny, bowiem wartość własna leży poza kołem jednostkowym. Brak oscylacji
